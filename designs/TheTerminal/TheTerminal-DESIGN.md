# DESIGN RATIONALE

## Core Concept

A utilitarian aesthetic for developers and technical audiences. It emphasizes data density and a "low-level" digital feel.

## UI Patterns

- **Monospace Everything:** All text, including navigation, must use monospaced fonts.
- **CRT Scanlines:** Apply a subtle horizontal line pattern over large images.
- **Modular Widgets:** Layout content in clearly defined boxes that look like separate terminal windows.
