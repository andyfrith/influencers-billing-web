# DESIGN RATIONALE

## Core Concept

Designed for investigative journalism and scientific deep-dives. This template prioritizes information density and factual authority.

## UI Patterns

- **Side-Note Footnotes:** Use a wide right margin where citations appear alongside the main body text.
- **Data Callouts:** Use monospaced blocks for statistics to distinguish them from editorial content.
- **Sticky TOC:** A persistent Table of Contents on the left side that highlights current progress.
