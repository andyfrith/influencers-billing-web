# DESIGN RATIONALE

## Core Concept

Inspired by Printed Circuit Boards (PCBs). Perfect for hardware reviews, DIY electronics, or robotics projects.

## UI Patterns

- **Trace Lines:** Use 45-degree and 90-degree lines as background decorative elements.
- **Solder-Pad Buttons:** Buttons should have circular "pads" at each corner.
- **Technical Overlays:** Use a persistent "System Status" bar that displays metadata about the page.
