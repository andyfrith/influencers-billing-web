# DESIGN RATIONALE

## Core Concept

Friendly, approachable, and organic. Ideal for wellness apps, gardening, or sustainability projects.

## UI Patterns

- **High Border Radius:** Everything from buttons to images should be heavily rounded (24px+).
- **Blob Shapes:** Use fluid, non-geometric shapes as background elements.
- **Pastel Palette:** Use low-saturation, earthy tones to keep the vibe calm.
