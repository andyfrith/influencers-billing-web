# DESIGN RATIONALE

## Core Concept

Built for fitness and sports. The layout should feel like it's in motion even when static.

## UI Patterns

- **Angled Dividers:** Use diagonal background shapes to create energy.
- **Heavy Typography:** Use thick, condensed sans-serifs for headings.
- **Sharp Corners:** 0px border-radii on all primary buttons to maintain an aggressive edge.
