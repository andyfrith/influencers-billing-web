# DESIGN RATIONALE

## Core Concept

Designed for historians, rare book collectors, or literary journals. It emphasizes the weight and texture of physical paper.

## UI Patterns

- **Deckled Edges:** Use CSS masks or SVGs to create "torn paper" edges on section dividers.
- **Sepia Tones:** All images should have a slight sepia or desaturated filter applied.
- **Footnote Focus:** Sidelined citations and small-print annotations are a primary layout feature.
