# DESIGN RATIONALE

## Core Concept

Optimized for social interaction, fan clubs, or neighborhood forums. It focuses on activity and density.

## UI Patterns

- **Channel Sidebar:** Persistent left-hand navigation for quick category switching.
- **Status Pills:** Small, rounded indicators for live users or trending topics.
- **Threaded Cards:** Content is nested in containers with subtle vertical connecting lines.
