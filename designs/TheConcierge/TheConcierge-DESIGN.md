# DESIGN RATIONALE

## Core Concept

A focused, step-by-step experience for high-end consulting. It removes all distractions to lead the user through a specific intake journey.

## UI Patterns

- **Progress Steppers:** A prominent progress bar at the top indicating remaining steps.
- **Selection Chips:** Large, clickable tiles for user choices instead of standard dropdowns.
- **Full-Screen Sections:** Each step occupies the entire viewport to keep the user focused on one question at a time.
