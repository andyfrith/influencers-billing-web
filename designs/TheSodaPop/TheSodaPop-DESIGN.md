# DESIGN RATIONALE

## Core Concept

High-contrast, high-energy, and playful. Best for candy shops, toy reviews, or youth-oriented social apps.

## UI Patterns

- **Rounded Blobs:** Use exaggerated, asymmetrical border-radii on all containers.
- **Sticker Effects:** Images should have a thick white border and a drop shadow to look like stickers.
- **Bouncy Animations:** Hover states should include a "scale-up" effect (`scale: 1.05`) with a bouncy easing function.
