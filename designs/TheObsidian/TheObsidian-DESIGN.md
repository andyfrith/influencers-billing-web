# DESIGN RATIONALE

## Core Concept

An ultra-minimalist, dark-first template for luxury tech, high-end watches, or private portfolios. It focuses on shadows and textures rather than colors.

## UI Patterns

- **True Black Background:** Use `#000000` to allow OLED screens to save power and provide infinite depth.
- **Inner Glows:** Instead of outer shadows, use subtle inner-border glows to define card shapes.
- **Low-Contrast Text:** Primary text should be off-white (`#999999`) to reduce eye strain in dark environments.
