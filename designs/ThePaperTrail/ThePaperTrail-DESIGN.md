# DESIGN RATIONALE

## Core Concept

Inspired by classic mid-century office stationery. Best for law offices, accountants, or professional administrative services.

## UI Patterns

- **Top-Tab Navigation:** Navigation items look like file folder tabs.
- **Subtle Rule Lines:** Use light gray horizontal lines (like notebook paper) for content separation.
- **Monotype Accents:** Use Courier or a similar font for document reference numbers.
