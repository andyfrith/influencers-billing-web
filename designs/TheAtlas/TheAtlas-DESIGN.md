# DESIGN RATIONALE

## Core Concept

Inspired by antique maps and globes. Perfect for travel agencies, history podcasts, or geopolitical blogs.

## UI Patterns

- **Compass Nav:** Use a circular or radial navigation menu in one corner.
- **Topo Gradients:** Use subtle topographic line patterns as background overlays.
- **Coordinate Breadcrumbs:** Use latitude/longitude formats for secondary page navigation or footers.
