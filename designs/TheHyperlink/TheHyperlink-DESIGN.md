# DESIGN RATIONALE

## Core Concept

A modern take on the early 2000s web directory. Extremely lightweight and optimized for mobile "link-in-bio" use cases.

## UI Patterns

- **Blue Links:** All primary interactive text is blue and underlined by default.
- **Button Stacks:** Full-width, stacked buttons with high contrast for mobile thumb-tapping.
- **Zero Icons:** Reliance on text hierarchy rather than visual cues to maximize load speed.
