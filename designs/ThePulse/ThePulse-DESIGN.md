# DESIGN RATIONALE

## Core Concept

A high-energy aesthetic for music festivals or live data dashboards. The design feels like it is "beating" with the content.

## UI Patterns

- **Radial Gradients:** Backgrounds radiate from the center rather than using linear colors.
- **Animated Outlines:** Buttons and card borders use a subtle glowing pulse effect.
- **Center-Weighted:** The most critical data is always dead-center in a circular layout.
