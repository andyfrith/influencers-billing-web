# DESIGN RATIONALE

## Core Concept

A professional upgrade to the original Canvas. It is designed for architecture firms and creative directors who need a structured but minimal presentation.

## UI Patterns

- **Fixed Aspect Ratios:** All images must maintain strict ratios (e.g., 4:3 or 16:9) to create a rhythmic grid.
- **Thin Vertical Borders:** Use vertical 1px lines to separate sidebar navigation from the main content.
- **Monochrome Icons:** Icons should be simple geometric shapes without any fill.
