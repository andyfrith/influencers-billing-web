# DESIGN RATIONALE

## Core Concept

For space enthusiasts, tech-forward research, or astronomy blogs. It uses "starlight" accents against deep navy.

## UI Patterns

- **Star-Field Background:** A subtle, fixed background image of stars.
- **Glassmorphism:** Use high-blur, low-opacity cards to simulate viewing through a lens.
- **Circular Progress:** Use radial bars for data visualization.
