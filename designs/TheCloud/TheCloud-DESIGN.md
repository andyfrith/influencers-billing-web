# DESIGN RATIONALE

## Core Concept

Weightless and futuristic. Perfect for cloud computing SaaS, weather apps, or flight-booking platforms.

## UI Patterns

- **Floating Elements:** Use large, soft shadows to make cards appear as if they are floating off the page.
- **Gradient Backgrounds:** Soft, vertical blue-to-white gradients that mimic the sky.
- **Rounded Clouds:** Use a maximum border-radius on buttons (50px+) to create "pill" or "cloud" shapes.
