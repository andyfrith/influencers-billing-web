# DESIGN RATIONALE

## Core Concept

For nightlife, high-tech streetwear, or futuristic tech reviews. It uses high-saturation accents against a void-black background.

## UI Patterns

- **Glow Borders:** Use CSS `box-shadow` to create neon tubes around cards.
- **Glitch Hover:** Text should have a subtle "glitch" or color-shift animation on hover.
- **Scanline Overlays:** A fixed, low-opacity PNG of horizontal lines over the entire viewport.
